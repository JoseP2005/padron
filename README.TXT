Ejercicio 10.2
El código define una clase llamada Habitante con tres variables privadas: nombre, apellido1 y apellido2. También tiene tres métodos públicos, setNombre, setApellido1 y setApellido2, para establecer el valor de cada una de las variables.

Ejercicio 11.4
Un método estático es un método que pertenece a la clase en lugar de a una instancia de la clase. Puede ser accedido directamente a través del nombre de la clase sin tener que crear un objeto. Se utiliza para operaciones o acceso a datos que no dependen de un estado específico del objeto y es útil para agrupar funciones relacionadas a nivel de clase.

Ejercicio 17.1
En Java los paquetes se definen mediante la estructura de directorios en la cual se encuentran los archivos fuentes de las clases, donde cada directorio representa un nivel en la estructura de paquetes.
